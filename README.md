# Rainfall-Agriculture-PowerBI
Power BI dashboard analyzing rainfall impact on agricultural productivity in India using rainfall and crop yield data (1966–2017).
Rainfall Impact on Agricultural Productivity Using Power BI
Project Overview

This project analyzes the impact of rainfall on agricultural productivity across Indian states using Power BI. The dashboard explores rainfall trends, crop yield performance, state-wise productivity, and rainfall correlation with agricultural output from 1966–2017.

Objectives
Analyze rainfall patterns across Indian states
Study the relationship between rainfall and crop productivity
Identify high and low performing agricultural regions
Build an interactive Power BI dashboard for insights and recommendations

Key Insights
Higher rainfall generally improves rice productivity
July and August contribute the highest rainfall
Punjab and Tamil Nadu show strong agricultural productivity
Some states maintain stable yields despite rainfall fluctuations
Recommendations
Improve irrigation systems in low rainfall areas
Promote climate-smart agriculture practices
Encourage water conservation methods
Use rainfall forecasting for crop planning
